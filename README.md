# niuh-spring-boot-starter
围绕SpringBoot开箱即用的Starter依赖，使得我们在开发业务代码时能够非常方便的、不需要过多关注框架的配置，而只需要关注业务即可

# 各 Module 介绍

| Module 名称                                                  | Module 介绍                                                  |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| [spring-boot-starter-redis](./spring-boot-starter-redis) | 基于Jedis的自定义redis-starter组件                               |
